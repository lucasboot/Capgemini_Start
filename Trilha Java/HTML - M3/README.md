### Tags estruturais semânticas do HTML5
- main: principal conteúdo, exclusivo
- header: cabeçalho
- footer: rodapé
- section: seção
- article: seção com significado próprio
- nav: navegação
- aside: não faz parte do conteúdo principal

#### Não semântico
- div: bloco
- span: pedaço
